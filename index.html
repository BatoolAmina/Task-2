API Documentation

Base URL

All API requests will be made to the following base URL:

https://api.example.com/api/v1/nudges

Create a Nudge (POST)

This endpoint is used to create a new nudge associated with a specific event or article.

Endpoint

/api/v1/nudges

Headers

Content-Type: application/json
Authorization: Bearer {your_api_key}

Payload

{
    "eventId": "event-123",
    "title": "Reminder: Upcoming Workshop",
    "imageUrl": "https://example.com/image.jpg",
    "scheduledOn": "2024-08-25T14:00:00Z",
    "description": "Don't miss the workshop on Reflections.",
    "invitationText": "Join us for a workshop on Reflections.",
    "icon": "reminder"
}

Response

{
    "nudgeId": "nudge-456",
    "eventId": "event-123",
    "title": "Reminder: Upcoming Workshop",
    "imageUrl": "https://example.com/image.jpg",
    "scheduledOn": "2024-08-25T14:00:00Z",
    "description": "Don't miss the workshop on Reflections.",
    "invitationText": "Join us for a workshop on Reflections.",
    "icon": "reminder"
}

Retrieve a Nudge (GET)

This endpoint retrieves details of a specific nudge using its ID.

Endpoint

/api/v1/nudges/{nudgeId}

Headers

Authorization: Bearer {your_api_key}

Response

{
    "nudgeId": "nudge-456",
    "eventId": "event-123",
    "title": "Reminder: Upcoming Workshop",
    "imageUrl": "https://example.com/image.jpg",
    "scheduledOn": "2024-08-25T14:00:00Z",
    "description": "Don't miss the workshop on Reflections.",
    "invitationText": "Join us for a workshop on Reflections.",
    "icon": "reminder"
}

Update a Nudge (PUT)

This endpoint updates the details of an existing nudge.

Endpoint

/api/v1/nudges/{nudgeId}

Headers

Content-Type: application/json
Authorization: Bearer {your_api_key}

Payload

{
    "title": "Updated Workshop Reminder",
    "imageUrl": "https://example.com/new-image.jpg",
    "scheduledOn": "2024-08-26T15:00:00Z",
    "description": "Updated: Don't miss the workshop on Reflections.",
    "invitationText": "Updated: Join us for a workshop on Reflections.",
    "icon": "reminder-new"
}

Response

{
    "nudgeId": "nudge-456",
    "eventId": "event-123",
    "title": "Updated Workshop Reminder",
    "imageUrl": "https://example.com/new-image.jpg",
    "scheduledOn": "2024-08-26T15:00:00Z",
    "description": "Updated: Don't miss the workshop on Reflections.",
    "invitationText": "Updated: Join us for a workshop on Reflections.",
    "icon": "reminder-new"
}

Delete a Nudge (DELETE)

This endpoint deletes a nudge by its ID.

Endpoint

/api/v1/nudges/{nudgeId}

Headers

Authorization: Bearer {your_api_key}

Response

204 No Content

List All Nudges (GET)

This endpoint retrieves a list of all nudges, with pagination options.

Endpoint

/api/v1/nudges

Headers

Authorization: Bearer {your_api_key}

Query Parameters

?type=latest&limit=10&page=1

Response

[

        "nudgeId": "nudge-456",

Operation Method Endpoint Description

Update a Nudge PUT /api/v1/nudges/{nudgeId} Update details of an existing nudge.
